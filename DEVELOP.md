# Logt

This part is about how to effectively use logt for development and production.

Some effective code example:
```cpp

int main(int argc, char** argv) {
    // 1. Initialize basic environment for logging only
    logt::claim("main");
    logt::addfile(...);
    logt::setFilterLevel(LogLevel::Debug);

    int result = 0; // You need to cache the return value of the workload anyway

    try {
        // This entire try block is for any workload logic, it also acts as
        // a lifetime scope for all objects created within. This makes sure
        // that any logs generated during construction/destruction of these
        // objects are handled.

        // And any exception would not crash the entire process, so we can
        // keep the logs that occurred before the exception.
    
        QApplication a(argc, argv);
        MainWindow m;
        m.show();
        result = a.exec();

    } catch (const std::exception &e) {
        logt.fatal() << "Runtime Error: " << e.what();
        result = -1;
    } catch (...) {
        logt.fatal() << "Unknown Critical Error";
        result = -2;
    }

    // 3. Clear and exit, make sure all logs are flushed and resources are released.
    logt::shutdown(); 
    return result;
}
```

> Also, you should always try to spread your workload logic into multiple functions, and keep your main function as clean as possible. It should just acts like the top-level layer that glues everything together.


#### Windows SEH (Structured Exception Handling)

On MSVC, __try/__except can be used to catch access violations.

So it can be used to catch some "uncaughtable" errors generated by the kernel, such as access violation, stack overflow, etc.

```cpp
__try {
    // your logic here
} __except(EXCEPTION_EXECUTE_HANDLER) {
    logt.fatal() << "Windows SEH caught a hard crash (e.g. Segfault)";
    logt::shutdown();
}
```


#### Stack traces

It is now possible to get the stack trace at the same time.

```cpp
#include <stacktrace>

try {
    // 业务逻辑
} catch (const std::exception& e) {
    logt.fatal() << "Exception: " << e.what();
    logt.fatal() << "Stacktrace:\n" << std::to_string(std::stacktrace::current());
}
```
