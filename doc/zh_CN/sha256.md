# sha256 - SHA256 哈希算法库

+ 名称: sha256  
+ 命名空间: `scl2`  
+ 文档版本: `1.1.0`

## CMake 配置信息

| 项目 | 值 |
|---------|---------|
| 命名空间 | `SharedCppLib2` |
| 库名称 | `sha256` |

包含方式:
```cmake
find_package(SharedCppLib2 REQUIRED)
target_link_libraries(target SharedCppLib2::sha256)
```

## 描述

`scl2::sha256` 提供 SHA-256 哈希功能（FIPS 180-4）。
输出为 256 位摘要（32 字节），通常可表示为 64 位十六进制字符串。

## 快速开始

### 基本文件哈希计算
```cpp
#include <SharedCppLib2/sha256.hpp>
#include <fstream>

int main() {
    std::ifstream ifs("file.txt", std::ios::binary);
    std::bytearray file_data;
    file_data.readAllFromStream(ifs);

    std::bytearray digest = scl2::sha256::getMessageDigest(file_data);
    std::cout << "SHA256: " << digest.toHex() << std::endl;
    return 0;
}
```

### 字符串哈希计算
```cpp
#include <SharedCppLib2/sha256.hpp>

std::string text = "Hello World";
std::string hex_hash = scl2::sha256::getHexMessageDigest(text);
std::cout << "哈希值: " << hex_hash << std::endl;
```

## 常见使用场景

### 文件完整性验证
```cpp
bool verify_file_integrity(const std::string& filename, const std::string& expected_hash) {
    std::ifstream file(filename, std::ios::binary);
    std::bytearray content;
    content.readAllFromStream(file);

    std::string actual_hash = scl2::sha256::getMessageDigest(content).toHex();
    return actual_hash == expected_hash;
}
```

### 数据比较
```cpp
bool compare_data(const std::bytearray& data1, const std::bytearray& data2) {
    return scl2::sha256::getMessageDigest(data1) == scl2::sha256::getMessageDigest(data2);
}
```
